add_training adds training data to the gaussian process appends the data if there already is some training data

Parameters
X- the input training data
y- list of discrete pairs [(dk, uk, vk), ...] dk = -1 if u > v, dk = 1 if v > u uk = index of the input (for the input set of points) vk = index of the second input @NOTE That this function updates the indicies if there is already training data. If inputing ordinal or abs data, it should be a vector of the same length as the input data (one y for each x)
type- type of input ['relative_discrete', 'ordinal', 'abs']
training_sigma- [opt] sets the uncertianty in the training data accepts scalars or a vector if each sample has a different uncertianty.
